### To have some text with color and other options: Show with a border

```bash
[user@server1]$ toilet -F border Cheat
┌───────────────────────────────────┐
│                                   │
│   mmm  #                      m   │
│ m"   " # mm    mmm    mmm   mm#mm │
│ #      #"  #  #"  #  "   #    #   │
│ #      #   #  #""""  m"""#    #   │
│  "mmm" #   #  "#mm"  "mm"#    "mm │
│                                   │
│                                   │
└───────────────────────────────────┘
```
