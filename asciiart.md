# Ascii Art

- To show some text in ASCII Art:

`figlet Cheat`

```sh
  ____ _                _
 / ___| |__   ___  __ _| |_
| |   | '_ \ / _ \/ _` | __|
| |___| | | |  __/ (_| | |_
 \____|_| |_|\___|\__,_|\__|

```

- To have some text with color and other options. Show with a border:

`toilet -F border Cheat`

```sh
┌───────────────────────────────────┐
│                                   │
│   mmm  #                      m   │
│ m"   " # mm    mmm    mmm   mm#mm │
│ #      #"  #  #"  #  "   #    #   │
│ #      #   #  #""""  m"""#    #   │
│  "mmm" #   #  "#mm"  "mm"#    "mm │
│                                   │
│                                   │
└───────────────────────────────────┘
```

- Basic show (filled):

`toilet Cheat`

```sh
   mmm  #                      m
 m"   " # mm    mmm    mmm   mm#mm
 #      #"  #  #"  #  "   #    #
 #      #   #  #""""  m"""#    #
  "mmm" #   #  "#mm"  "mm"#    "mm
```
